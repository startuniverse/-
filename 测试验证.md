## 如果仍有问题

如果重启后仍有问题，请检查：

1. **编译是否成功**：`mvn compile` 应显示 BUILD SUCCESS
2. **应用是否重启**：确保使用的是新编译的代码
3. **查看控制台日志**：是否有错误信息
4. **检查数据库**：新增的数据是否已插入

## 数据库验证命令

```bash
# 查看学生+用户关联数据
mysql -u root -p123456 education_platform -e "
SELECT s.id, s.student_number, u.real_name as name, s.class_id
FROM student s LEFT JOIN user u ON s.user_id = u.id;
"

# 查看资产数据
mysql -u root -p123456 education_platform -e "
SELECT id, asset_code, asset_name, category, quantity, school_id
FROM office_asset;
"
```
