## 🚀 部署方案

### 开发环境
```bash
# 后端
mvn spring-boot:run

# 前端
npm run dev
```

### 生产环境
```bash
# 1. 打包后端
mvn clean package
java -jar target/education-platform-1.0.0.jar

# 2. 构建前端
npm run build

# 3. Nginx部署
# 配置见 DEPLOYMENT.md
```

### Docker部署
```bash
# 后端容器
docker build -t edu-backend .
docker run -d -p 8080:8080 edu-backend

# 前端容器
docker build -t edu-frontend .
docker run -d -p 80:80 edu-frontend
```
