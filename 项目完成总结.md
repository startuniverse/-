## 🔄 完整业务流程

### 场景1：管理员创建课程
```
1. 登录管理员账号
2. 进入"培训管理"
3. 点击"新增课程"
4. 填写表单：
   - 课程标题：教师信息技术培训
   - 讲师：张老师
   - 分类：专业技能
   - 科目：综合
   - 难度：中级
   - 时长：120分钟
5. 选择"已发布"
6. 点击保存
7. 查看统计面板（课程数+1）
```

### 场景2：学生报名学习
```
1. 登录学生账号
2. 进入"培训课程"
3. 搜索"信息技术"
4. 点击课程卡片查看详情
5. 点击"立即报名"
6. 进入"我的培训"
7. 点击"继续学习"（模拟）
8. 点击"考试"
9. 回答4道题目
10. 提交试卷
11. 查看得分（自动计算）
12. 查看"我的证书"
```
