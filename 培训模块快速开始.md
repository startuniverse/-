# 在线学习培训模块 - 快速开始指南

## 🚀 5分钟快速启动

### 第1步：创建数据库表

```bash
# 连接到MySQL
mysql -u root -p

# 创建数据库（如果不存在）
CREATE DATABASE IF NOT EXISTS education_platform CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 退出MySQL
exit

# 导入表结构
mysql -u root -p education_platform < E:\shixun\database\schema.sql
```

**验证：** 登录MySQL，执行 `SHOW TABLES;` 应该看到8张training_开头的表

---

### 第2步：启动后端服务

```bash
# 打开命令行，进入后端目录
cd E:\shixun\backend

# 启动Spring Boot应用
mvn spring-boot:run
```

**验证：**
- 看到 `Started EducationPlatformApplication` 表示启动成功
- 访问 `http://localhost:8080` 应该有响应

---

### 第3步：启动前端服务

```bash
# 打开新的命令行窗口，进入前端目录
cd E:\shixun\frontend

# 安装依赖（首次运行需要）
npm install

# 启动开发服务器
npm run dev
```

**验证：**
- 看到 `Local: http://localhost:5173` 表示启动成功
- 浏览器访问 `http://localhost:5173`

## 🔧 常见问题

### Q1: 数据库连接失败
**错误：** `Communications link failure`

**解决：**
```bash
# 检查MySQL是否启动
net start mysql

# 检查配置文件
# E:\shixun\backend\src\main\resources\application.yml
# 确认数据库连接信息正确
```

### Q2: 前端无法访问后端
**错误：** `Network Error` 或 `404`

**解决：**
1. 检查后端是否启动（端口8080）
2. 检查前端配置：`E:\shixun\frontend\src\utils\request.js`
3. 确认baseURL正确：`http://localhost:8080`

### Q3: 菜单不显示培训相关选项
**原因：** 路由或菜单未更新

**解决：**
1. 检查路由文件：`E:\shixun\frontend\src\router\index.js`
2. 检查Layout文件：`E:\shixun\frontend\src\views\Layout.vue`
3. 刷新浏览器缓存：Ctrl+F5

### Q4: 报名或考试失败
**错误：** `权限不足` 或 `用户信息为空`

**解决：**
1. 确保已登录
2. 检查localStorage是否有token和userInfo
3. 查看浏览器控制台错误信息
